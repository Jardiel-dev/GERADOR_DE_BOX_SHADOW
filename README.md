# 🎨 Gerador de Box Shadow

Aplicação web que permite gerar sombras CSS (`box-shadow`) de forma visual e interativa, facilitando o desenvolvimento de interfaces modernas.

Link: 

---

## 🚀 Funcionalidades

* Ajuste em tempo real dos parâmetros da sombra:

  * Deslocamento horizontal e vertical
  * Blur (borrão)
  * Spread (expansão)
  * Cor e opacidade
  * Opção de sombra interna (inset)

* Preview instantâneo do efeito aplicado

* Geração automática do código CSS

* Compatibilidade com diferentes navegadores (`-webkit` e `-moz`)

* Copiar código com apenas um clique

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📚 Sobre o projeto

Este projeto foi desenvolvido durante o curso **Formação Front-End** da Udemy, com o objetivo de praticar conceitos fundamentais de desenvolvimento web como manipulação do DOM, eventos e estilização com CSS.

Além disso, foram aplicadas melhorias na organização do código e boas práticas para torná-lo mais próximo de um ambiente profissional.
